Getting Started

01

Enable Debugging

Turn on USB Debugging in your phone's Developer Options.

02

Plug in & Trust

Connect via USB. When prompted, tap Allow on your screen to authorize your computer.

03

Enjoy Gaze

Launch the app. It will auto-detect your device. Start mirroring or managing instantly.

Features

  • Wireless & Unified Devices

    Connect via IP/Port pairing with mDNS support. Manage physical and wireless devices (USB/Wi-Fi merging) in a single, unified view with device pinning and history support.

  • Advanced File Manager

    Full-featured explorer with drag-drop upload, seamless downloads, and easy file manipulation (Copy/Cut/Paste/Delete).

  • Scrcpy Mirroring & Recording

    Low-latency mirroring with audio forwarding. Support for simultaneous recording of multiple devices in the background with visual tray indicators and one-click access to recording folders.

  • Total App Control

    One-click install/uninstall, APK export, data clearing, and detailed package management.

  • Developer Tools

    Powerful Logcat with pre-filtering rules and integrated ADB Shell console for direct command execution.

  • Proxy & Network Sniffer

    Integrated HTTP/HTTPS proxy with MITM decryption, WebSocket support, and real-time traffic shaping (bandwidth limiting & latency simulation).

Troubleshooting

macOS: "App is damaged and can't be opened"

If you see this error, it's due to macOS Gatekeeper quarantine on downloaded unsigned apps. To fix it, run this command in your terminal:

sudo xattr -cr /path/to/Gaze.app

*Replace /path/to/adbGUI.app with the actual path to your downloaded application (e.g., /Applications/Gaze.app)*

Windows: "Windows protected your PC"

If Microsoft Defender SmartScreen prevents startup:
Click More info → Run anyway.

✨ Alternative: You can also build from source safely on your own machine.
