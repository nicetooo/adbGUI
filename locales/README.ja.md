# ADB GUI 🚀

**Wails**、**React**、**Ant Design** で構築された、強力でモダンな自己完結型の Android 管理ツールです。

> ✨ **注意**: このアプリケーションは純粋な **vibecoding** の産物です。

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-macOS%20%7C%20Windows%20%7C%20Linux-lightgrey.svg)

## ✨ 特徴

### 📱 デバイス管理
- 接続されたデバイスのリアルタイム監視。
- デバイス ID、モデル、接続状態の表示。
- アプリ、シェル、Logcat、ミラーリングへのワンクリックアクセス。

### 📦 アプリ管理
- インストールされているすべてのパッケージ（システムアプリおよびユーザーアプリ）をリスト表示。
- 名前またはタイプでアプリをフィルタリングおよび検索。
- **アクション**: 強制停止、データ消去、有効化/無効化、アンインストール。
- **クイック Logcat**: アプリリストから特定のアプリのログに直接ジャンプ。

### 🖥️ スクリーンミラーリング (Scrcpy)
- **内蔵 Scrcpy**: 外部に何もインストールする必要はありません。
- きめ細かな制御:
  - ビデオビットレートと最大 FPS。
  - 解像度（最大サイズ）。
  - スリープ防止と画面オフオプション。
  - 常に手前に表示。
  - オーディオストリーミングの切り替え。

### 📜 高度な Logcat
- 自動スクロール付きのリアルタイムログストリーミング。
- **アプリ固有のフィルタリング**: 特定のパッケージ名でログをフィルタリング。
- **自動監視**: アプリが開く前にログ記録を開始。ツールは自動的に PID を検出し、アプリが起動するとフィルタリングを開始します。
- キーワード検索/フィルタリング。

### 💻 ADB シェル
- ADB コマンドを実行するための統合ターミナル。
- 出力履歴付きのクイックコマンド実行。

---

## 🛠️ 内蔵バイナリ

このアプリケーションは完全に自己完結型です。以下を同梱しています：
- **ADB** (Android Debug Bridge)
- **Scrcpy** 実行ファイル
- **Scrcpy-server**

起動時に、これらは一時ディレクトリに展開され、自動的に使用されます。システムの PATH を設定する必要はありません。

---

## ⚠️ Xiaomi/Poco/Redmi ユーザーへの重要な注意

Scrcpy で**タッチ操作**を有効にするには、以下を行う必要があります：
1. **開発者オプション**に移動します。
2. **USB デバッグ**を有効にします。
3. **USB デバッグ（セキュリティ設定）**を有効にします。
   *(注意: ほとんどの Xiaomi デバイスでは、これに SIM カードと Mi アカウントのログインが必要です)*。

## 🚀 はじめに

### 前提条件
- **Go** (v1.21+)
- **Node.js** (v18+)
- **Wails CLI** (`go install github.com/wailsapp/wails/v2/cmd/wails@latest`)

### 開発
```bash
wails dev
```

### ビルド
```bash
wails build
```
コンパイルされたアプリケーションは `build/bin` で利用可能です。

### リリース
このプロジェクトは GitHub Actions を使用して、マルチプラットフォームのビルドを自動化しています。新しいリリースを作成するには：
1. コミットにタグを付ける: `git tag v1.0.0`
2. タグをプッシュする: `git push origin v1.0.0`
GitHub Action は macOS、Windows、Linux 用に自動的にビルドし、アーティファクトをリリースページにアップロードします。
